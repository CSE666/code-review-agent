# 🚀 Multi-Agent Code Review System

A multi-agent LLM-powered system for automated code review, bug detection, and self-healing via test-driven feedback loops.

---

## 📌 Overview

This project implements an **AI-driven code review pipeline** using a **multi-agent architecture**.  
It automatically analyzes source code, identifies issues, generates fixes, and validates them through unit testing.

The system is designed to reduce manual code review effort and improve development efficiency.

---

## 🧠 Key Features

- 🔍 **Automated Code Analysis**
  - Detects bugs, logic errors, and edge cases
  - Identifies performance issues and code smells

- 🛠 **Auto Code Fixing**
  - Generates corrected code based on analysis
  - Maintains original logic consistency

- 🧪 **Test-Driven Validation**
  - Runs unit tests automatically using `pytest`
  - Provides feedback for iterative improvement

- 🔁 **Closed-loop Pipeline**
  - Analysis → Fix → Test → Feedback

- 🤖 **Multi-Agent Collaboration**
  - Analysis Agent
  - Fix Agent
  - Test Agent
